# Groq

Groq delivers ultra-low-latency inference via its LPU (Language Processing Unit) hardware. It is OpenAI API-compatible, making integration straightforward.

<Tip>
  Groq is notably faster than GPU-based cloud providers for tokens-per-second throughput — ideal for latency-sensitive agent tasks.
</Tip>

## Prerequisites

Create an API key at [console.groq.com](https://console.groq.com) and set it on the gateway host:

```bash
echo 'GROQ_API_KEY=your-key-here' >> ~/.openclaw/.env
```

## Configuration

```json5
{
  models: {
    providers: {
      groq: {
        apiKey: "GROQ_API_KEY", // references the env var
        models: [
          "llama-4-scout-17b-16e-instruct",
          "llama-4-maverick-17b-128e-instruct",
          "llama-3.3-70b-versatile",
          "llama-3.1-8b-instant",
          "deepseek-r1-distill-llama-70b",
          "kimi-k2-instruct",
          "qwen-qwq-32b",
          "qwen3-32b",
          "mistral-saba-24b",
          "gemma2-9b-it",
        ],
      },
    },
  },
}
```

<Note>
  `apiKey` is the name of the environment variable, not the key value itself. Keep secrets in `.env`, not in committed config files.
</Note>

## Selecting a model

```json5
{
  agents: {
    defaults: {
      model: { primary: "groq/llama-3.3-70b-versatile" },
    },
  },
}
```

| Model | Notes |
| ----- | ----- |
| `llama-4-scout-17b-16e-instruct` | Llama 4 Scout — efficient 17B MoE |
| `llama-4-maverick-17b-128e-instruct` | Llama 4 Maverick — 128-expert MoE |
| `llama-3.3-70b-versatile` | Best general-purpose Llama 3.3 model on Groq |
| `llama-3.1-8b-instant` | Fastest — lowest latency |
| `deepseek-r1-distill-llama-70b` | DeepSeek R1 distilled into Llama 70B |
| `kimi-k2-instruct` | Moonshot Kimi K2 — strong coding |
| `qwen-qwq-32b` | Qwen QwQ reasoning model |
| `qwen3-32b` | Qwen 3 32B |
| `mistral-saba-24b` | Mistral Saba 24B |
| `gemma2-9b-it` | Google Gemma 2 9B instruction-tuned |

<Note>
  The full model list is auto-discovered from Groq's API when a key is configured. Run `openclaw models list --provider groq` for the current catalog — Groq's offerings change frequently.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    The API key is invalid or revoked. Generate a new one at [console.groq.com](https://console.groq.com) and update `~/.openclaw/.env`.
  </Accordion>
  <Accordion title="429 Rate limited">
    Groq free-tier limits are generous but finite. Check your usage at the Groq console or add a paid plan.
  </Accordion>
  <Accordion title="Model not available">
    Groq's model catalog changes frequently. Run `openclaw models list --provider groq` for the current list.
  </Accordion>
</AccordionGroup>
