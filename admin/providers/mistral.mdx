# Mistral AI

WednesdayAI supports Mistral AI's model family, including frontier and open-weight models.

## Prerequisites

Create an API key at [console.mistral.ai](https://console.mistral.ai) and set it on the gateway host:

```bash
echo 'MISTRAL_API_KEY=your-key-here' >> ~/.openclaw/.env
```

## Configuration

Add the provider under `models.providers` in your config:

```json5
{
  models: {
    providers: {
      mistral: {
        apiKey: "MISTRAL_API_KEY", // references the env var
        models: [
          "mistral-large-latest",
          "mistral-small-latest",
          "codestral-latest",
          "mistral-nemo",
          "magistral-medium-latest",
          "magistral-small-latest",
          "devstral-small-2507",
          "pixtral-large-latest",
          "ministral-3b-latest",
          "ministral-8b-latest",
          "mistral-embed",
        ],
      },
    },
  },
}
```

<Note>
  `apiKey` is the name of the environment variable, not the key value itself. Keep secrets in `.env`, not in committed config files.
</Note>

## Selecting a model

```json5
{
  agents: {
    defaults: {
      model: { primary: "mistral/mistral-large-latest" },
    },
  },
}
```

| Model | Notes |
| ----- | ----- |
| `mistral-large-latest` | Flagship — best reasoning |
| `mistral-small-latest` | Fast and cost-efficient |
| `codestral-latest` | Code-specialised |
| `mistral-nemo` | Lightweight open-weight model |
| `magistral-medium-latest` | Magistral — strong reasoning |
| `magistral-small-latest` | Magistral small — fast reasoning |
| `devstral-small-2507` | Code-focused agentic model |
| `pixtral-large-latest` | Multimodal — vision + text |
| `ministral-3b-latest` | Ultra-lightweight edge model |
| `ministral-8b-latest` | Edge-optimised 8B model |
| `mistral-embed` | Text embeddings |

Run `openclaw models list --provider mistral` to see the current catalog.

<Note>
  **Audio transcription:** `voxtral-mini-latest` supports audio input via the `tools.media.audio` config key.

  **Vector embeddings:** `mistral-embed` produces dense text embeddings suitable for semantic search and RAG pipelines. Route it with the `mistral/mistral-embed` model ID.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    The API key is invalid or expired. Generate a new key at [console.mistral.ai](https://console.mistral.ai) and update `~/.openclaw/.env`.
  </Accordion>
  <Accordion title="429 Rate limited">
    You have exceeded your tier limits. Check your usage and quota at the Mistral console.
  </Accordion>
  <Accordion title="Model not found">
    Verify the model ID is correct and that your subscription plan includes access to it.
  </Accordion>
</AccordionGroup>
