# Together AI

Together AI provides scalable inference for open-weight models via an OpenAI-compatible API. It hosts a broad catalog of Llama, Qwen, Mistral, and other community models.

## Prerequisites

Create an API key at [api.together.ai](https://api.together.ai) and set it on the gateway host:

```bash
echo 'TOGETHER_API_KEY=your-key-here' >> ~/.openclaw/.env
```

## Configuration

Together AI's endpoint is OpenAI-compatible. Set the base URL and API key:

```json5
{
  models: {
    providers: {
      together: {
        apiKey: "TOGETHER_API_KEY", // references the env var
        models: [
          "zai-org/GLM-4.7",
          "meta-llama/Llama-4-Scout-17B-16E-Instruct",
          "meta-llama/Llama-4-Maverick-17B-128E-Instruct-Turbo",
          "deepseek-ai/DeepSeek-V3.1",
          "deepseek-ai/DeepSeek-R1",
          "moonshotai/Kimi-K2-Instruct-0905",
        ],
      },
    },
  },
}
```

<Note>
  `apiKey` is the name of the environment variable, not the key value itself. Keep secrets in `.env`, not in committed config files.
</Note>

<Note>
  Together AI's API base URL is `https://api.together.xyz/v1`. This is set automatically for the `together` provider — no `baseUrl` override is needed.
</Note>

## Selecting a model

```json5
{
  agents: {
    defaults: {
      model: { primary: "together/zai-org/GLM-4.7" },
    },
  },
}
```

| Model | Notes |
| ----- | ----- |
| `zai-org/GLM-4.7` | GLM-4.7 — current post-onboard default |
| `meta-llama/Llama-4-Scout-17B-16E-Instruct` | Llama 4 Scout 17B — efficient mixture-of-experts |
| `meta-llama/Llama-4-Maverick-17B-128E-Instruct-Turbo` | Llama 4 Maverick 17B — large MoE, Turbo variant |
| `deepseek-ai/DeepSeek-V3.1` | DeepSeek V3.1 — strong general-purpose |
| `deepseek-ai/DeepSeek-R1` | DeepSeek R1 — extended reasoning |
| `moonshotai/Kimi-K2-Instruct-0905` | Kimi K2 Instruct — September 2025 snapshot |

<Note>
  Together AI hosts hundreds of open models. The runtime discovers available models automatically when a key is configured. Run `openclaw models list --provider together` for the full current catalog.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    The API key is invalid or revoked. Generate a new one at [api.together.ai](https://api.together.ai) and update `~/.openclaw/.env`.
  </Accordion>
  <Accordion title="Model not found">
    Together AI model IDs are case-sensitive and include the vendor prefix. Check the full ID in the Together AI model catalog.
  </Accordion>
  <Accordion title="429 Rate limited">
    You have exceeded request limits. Check your plan and usage at the Together AI console.
  </Accordion>
</AccordionGroup>
